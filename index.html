How It Works
1
GitHub Actions fetches prices
Runs every 15 min, Mon–Fri. Python fetches Yahoo Finance server-side, writes prices.json to your repo.
2
App fetches same-origin JSON
Your app at murugkan.github.io/bharat-markets fetches ./prices.json — same domain, zero CORS on Safari.
3
Add any NSE symbol
Type any NSE symbol in search — not just the preset list. Also add it to watchlist.txt in your repo so the Action fetches it.
.github/workflows/fetch-prices.yml
name: Fetch NSE Prices on: schedule: - cron: '*/15 3-10 * * 1-5' workflow_dispatch: jobs: fetch: runs-on: ubuntu-latest permissions: contents: write steps: - uses: actions/checkout@v4 - uses: actions/setup-python@v5 with: python-version: '3.12' - run: pip install yfinance - run: python fetch_prices.py - run: | git config user.name "github-actions[bot]" git config user.email "github-actions[bot]@users.noreply.github.com" git add prices.json git diff --staged --quiet || git commit -m "chore: update prices" git push
