How It Works
1
GitHub Actions fetches prices
A scheduled workflow runs every 15 min during NSE market hours. It calls Yahoo Finance server-side (Python), writes prices.json to your repo root, and commits it.
2
App reads the same-origin JSON
Your iPhone app at username.github.io/repo fetches ./prices.json — same domain, zero CORS. Works perfectly on Safari.
3
Manual refresh reads latest JSON
Tap the ↻ button to re-fetch prices.json. If offline, the app uses the last cached version from localStorage.
Repo File Structure
Your GitHub repo must have these files:
your-repo/ ├── index.html # ← this app ├── prices.json # ← auto-generated by Action └── .github/ └── workflows/ └── fetch-prices.yml
.github/workflows/fetch-prices.yml
Create this file in your repo. It runs every 15 min on weekdays between 9:00–15:45 IST (3:30–10:15 UTC).
name: Fetch NSE Prices on: schedule: # Every 15 min, Mon–Fri, 3:30–10:15 UTC (= 9:00–15:45 IST) - cron: '*/15 3-10 * * 1-5' workflow_dispatch: # Allow manual trigger jobs: fetch: runs-on: ubuntu-latest permissions: contents: write steps: - uses: actions/checkout@v4 - name: Set up Python uses: actions/setup-python@v5 with: python-version: '3.12' - name: Install dependencies run: pip install yfinance requests - name: Fetch prices run: python fetch_prices.py - name: Commit prices.json run: | git config user.name "github-actions[bot]" git config user.email "github-actions[bot]@users.noreply.github.com" git add prices.json git diff --staged --quiet || git commit -m "chore: update prices $(date -u +%H:%M)" git push
fetch_prices.py
Place this in your repo root. It reads watchlist.txt (one symbol per line, no .NS) and outputs prices.json.
import yfinance as yf, json, os from datetime import datetime, timezone # Read symbols from watchlist.txt or use defaults try: with open("watchlist.txt") as f: syms = [l.strip().upper() for l in f if l.strip()] except FileNotFoundError: syms = ["RELIANCE", "TCS", "INFY", "HDFCBANK", "ICICIBANK"] ns_syms = [s + ".NS" for s in syms] tickers = yf.Tickers(" ".join(ns_syms)) out = {} for raw_sym, sym in zip(ns_syms, syms): try: t = tickers.tickers[raw_sym] inf = t.fast_info hi = t.history(period="1y", interval="1d") out[sym] = { "ltp": round(inf.last_price, 2), "prev": round(inf.previous_close, 2), "change": round(inf.last_price - inf.previous_close, 2), "changePct": round((inf.last_price - inf.previous_close) / inf.previous_close * 100, 3), "open": round(inf.open, 2), "high": round(inf.day_high, 2), "low": round(inf.day_low, 2), "vol": inf.three_month_average_volume, "w52h": round(inf.year_high, 2), "w52l": round(inf.year_low, 2), "mktCap": inf.market_cap, "ma50": round(hi["Close"].tail(50).mean(), 2) if len(hi)>=50 else None, "ma200": round(hi["Close"].tail(200).mean(), 2) if len(hi)>=200 else None, "name": t.info.get("longName", sym), "pe": t.info.get("trailingPE"), "fwdPe": t.info.get("forwardPE"), "pb": t.info.get("priceToBook"), "divYield": t.info.get("dividendYield"), "eps": t.info.get("trailingEps"), "bv": t.info.get("bookValue"), "avgVol": t.info.get("averageVolume"), } except Exception as e: print(f" ⚠ {sym}: {e}") payload = { "updated": datetime.now(timezone.utc).isoformat(), "quotes": out, } with open("prices.json", "w") as f: json.dump(payload, f, indent=2) print(f"✓ Wrote {len(out)} quotes to prices.json")
watchlist.txt (optional)
One NSE symbol per line (no .NS suffix). The Action will auto-sync this from what you add in the app on next run.
